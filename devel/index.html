Compiling nmag libraries and nmag from source

We assume that you are reasonable familiar with the install procedure. If not, look it up in installation instructions (method A).

Download the latest overall tar file nmag-devel.tar.gz.

Updating the nmag-core only

Assuming you have downloaded nmag-devel.tar.gz before, and you just want an update of the nmag core (not the other libraries such as python, petsc, qhull, ...), then:

  1. download nmag-devel-core.tar.gz

  2. go to the parent directory of nmag-devel and untar:

    $ tar xfvz nmag-devel-core.tar.gz

  3. Update the core:

    $ cd nmag
$ . exports.bash
$ cd nsim
$ rm bin/nsim
$ make
